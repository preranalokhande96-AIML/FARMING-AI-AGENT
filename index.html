📋 Quick Setup Steps
1
Create IBM Cloud account

Go to cloud.ibm.com → Sign up (free tier available) → Open Watsonx.ai Studio

2
Get your API Key

IBM Cloud → Manage → Access (IAM) → API keys → Create an IBM Cloud API key

IBM_API_KEY=your_key_here
3
Get your Project ID

Watsonx Studio → Create/open a Project → Settings → General → copy the Project ID

IBM_PROJECT_ID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
4
Edit your .env file

Open .env in your project folder and fill in both values, then restart the Flask server.

IBM_API_KEY=your_actual_api_key
IBM_PROJECT_ID=your_actual_project_id
IBM_WATSONX_URL=https://us-south.ml.cloud.ibm.com
5
Restart Flask & test

Stop the server (Ctrl+C) and restart: python app.py

The status badge will turn ● Live when connected.
